# 证件扫描（Card Scan）

把放在 HP DeskJet 5135 平板扫描仪上的证件，通过「扫正面 → 翻面 → 扫反面」两次扫描，合成为一张 A4 复印件页的小工具。

## Language

**证件（Document）**:
需要复印正反两面的实体卡/证。面积不超过半张 A4（A5 以内），正反两面尺寸一致。
_Avoid_: 卡片、文件

**正面 / 反面（Front / Back）**:
证件的两个物理面。正面指用户第一次扫描的那一面，反面指翻转后第二次扫描的那一面。

**原始扫描图（Raw Scan）**:
扫描仪对整块玻璃板拍下的完整图像，含证件和盖板背景。

**证件图（Card Image）**:
从原始扫描图中自动找边、裁剪、摆正后得到的图像，含证件本身及四周约 3mm 的自然背景余量。

**复印件页（Composite Page）**:
最终产物：一页 A4 纵向 PDF，正面证件图居中放上半页、反面居中放下半页，按 1:1 物理真实尺寸渲染；页面底色取自原始扫描图的背景采样色，整页观感如一张真实复印件。
_Avoid_: 合并页、输出页

**扫描设备（Device）**:
局域网内通过 mDNS（`_uscan._tcp`）发现的 eSCL 扫描仪。系统记住上一次使用的设备作为默认；不可用时需重新选择。
